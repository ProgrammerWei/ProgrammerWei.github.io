Veröffentlicht am 2018-01-21

1、下载hadoop

地址：http://mirrors.tuna.tsinghua.edu.cn/apache/hadoop/common/，解压。

2、下载windows环境文件

地址：http://download.csdn.net/download/u010934981/10216554，覆盖hadoop-3.0.0\bin，并将hadoop.dll复制一份到c:/windows/System32。

3、配置hadoop环境变量

创建HADOOP_HOME，在Path下添加 %HADOOP_HOME%\bin
